What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a structured representation of an HTML document, organized as a tree of nodes (elements, attributes, text).

How do you select an element from the DOM?

You can select elements from the DOM using various JavaScript methods. Here are some common ways:
1.document.getElementById()
2.document.getElementsByClassName()
3.document.getElementsByTagName()
4.document.querySelector()
5.document.querySelectorAll()

What is event delegation in the context of the DOM, and is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events from its child elements. It works through event bubbling, allowing the parent to capture events triggered by children. This method improves performance, simplifies code, and easily handles dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using JavaScript in the DOM as follows:
1. Manipulating Attributes:
1. setAttribute()
2. getAttribute()
3. removeAttribute()

2. Manipulating Styles:
1. Using style property: Directly modify an element's inline styles.
element.style.color = "blue";
element.style.backgroundColor = "yellow";
2. Add/Remove CSS classes: Use the classList property to manage CSS classes.
element.classList.add("newClass");
element.classList.remove("oldClass");
element.classList.toggle("active");
